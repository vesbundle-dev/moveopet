# MoveoPet

Статичен лендинг за продукт за подвижни стави при кучета.

## Файлове
- `index.html`
- `styles.css`
- `script.js`
- `product.png`

## Пускане локално
Отвори `index.html` в браузър.

## Качване в GitHub + Vercel
1. Създай ново repository: `moveopet`
2. Качи всички файлове
3. Във Vercel избери **Add New Project**
4. Import repo-то
5. Deploy
